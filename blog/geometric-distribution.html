Geometric Distribution

18/07/2022

Use Case

The Geometric Distribution is used to calculate how many events / failures will occur until we get one success
Note: The geometric distribution is just the negative binomial distribution with the number of successes r = 1


X ~ Geo(p), 0 < p < 1
p = probability of event occurring
i = number of events until one success

Pr(X = i), p(i)

p * ((1 - p)^(i - 1))

Pr(X <= i), F(i)

1 - (1 - p)^i

Expected Value (E[X])

1 / p

Variance (Var[X])

(1 - p) / (p^2)

Negative Binomial Distribution

18/07/2022

Use Case

The Negative Binomial Distribution is used to calculate how many events / failures will occur until we get r number of success.

This distribution is used if we have independent trials and perform them until we get a specific number of successes.

X ~ NB(r, p), 0 < p < 1
r = success trials
p = probability of the event occurring once
i = number of trials / failures until r successes

Pr(X = i), p(i)

Pr(X <= i), F(i)

Expected Value (E[X])

r / p

Variance (Var[X])

(r(1 - p)) / (p^2)

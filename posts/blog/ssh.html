How to Connect to Device Shell Using SSH

19/07/2022

Description

SSH (secure shell) allows you to remotely connect to a computer or device's terminal / shell session. This is typically useful for sysadmins, and provides a way to access a device's filesystem, or to run commands on a device without having physical access.

How to Use

Run SSH From a Linux or MacOS Device and Remotely Connect to Another Device
$ ssh username@hostname
or
$ ssh hostname -l username

e.g.
$ ssh pi@192.168.0.24


CLA Breakdown

ssh The SSH program installed by default on most Linux, and MacOS distributions

-l Specifies the local account username on the remote machine. SSH will then prompt the for the corrosponding password of this account.

Note: The login credentials you will supply in SSH is for logging into an account present on the machine you are connecting to. The credentials used on your machine will not work when using SSH to log into another device.

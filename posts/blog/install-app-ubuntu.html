How to Install an Application on Ubuntu/Debian

05/08/2022

Description

Debian is a Linux based distribution at the heart of the most popular operating systems today. It powers operating systems such a Ubuntu, Linux mint, KDE Neon, Kubuntu, Xububtu, elementary os, and Deepin just to name a few. One of the main selling points and innovations of Debian based distributions is it's clear and easy to use application installer / app management. The package tool used in many Debian based distros is APT.

How to Use

Installing From Package Repository

To install an application such as NodeJS with APT simple run the following command from the terminal window:

$ sudo apt install nodejs -y
Note: the -y flag is optional, but it does remove an extra conformation step otherwise required before you install an application.

To download an application on APT, you may need to add some extra app repositories or find the package name of the application. To do both simple search for the program you want to download + “apt download”. The download instructions may specify Debian or Ubuntu, or another operating system that you are not running. However, if you are running APT on your system, you will be able to follow the instructions whether or not the instructions are specific for your operating system.

Note: for Ubuntu22.0+ there may be a separate set of install instructions you will need to follow

Installing from .deb package file

$ sudo apt install ./NodeJS.deb

You will have to replace ./NodeJs.deb with the relative path to where your terminal is running from. To check the current path your terminal is running from, use the command pwd.


Command Line Argument Breakdown

sudo - installs the application at the root directory, this means that all users will be able to use the application. Without sudo, only your user account will be able to use the application. Note that some applications MUST be run as root, and will not run without sudo.

apt - is the command name for the package manager where we are going to get our application from.

install - tells APT to install the proceeding application.

NodeJS - you can replace this with the package name of the program you are trying to install. This is the program that will install.

-y flag - do not prompt for conformation before install. Not using this flag may be useful if you have limited disk space as it prompts you to confirm the disk size and package repository before install

How to Extract / Open .tar.gz File

19/07/2022

Description

.tar.gz files are compressed files and archives that are used to store data, and is primarily used by Linux / MacOS platforms for web downloads. Tar or "tape archiver" is a file format which combines multiple or directory files into a single file. gz or "gzip" is a compression format which is used to compress a singular file. Therefore, for this reason, tar is used to combine a directory into a single file, while gz is used to compress and reduce the size of the file.

Problem

After downloading a file with a filename similar to file.tar.gz, you are unable to open it.

Solution

Extract the fileName.tar.gz File Using the CLI
$ tar -xvzf fileName.tar.gz


CLA Breakdown

tar The tar program installed by default on most Linux, and MacOS distributions

-x Extracts files from an archive (specifies that the program should extract files)

-v Displays verbose / debug info

-z Filters the archive through the separate gzip program

-f Specifies in file name / path

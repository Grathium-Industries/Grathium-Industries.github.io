Hello World in Many Programing Languages

31/07/2022

Description

Hello world programs are typically the first program that a new programer creates to show the basic syntax of a programming language. The goal of this article is to show how to write a hello world program in as many programming languages as possible (or just keep track of all the languages I can vaguely program in).

C

#import <stdio.h>

int main() {
printf("Hello World!\n");
return 0;
}

C++

#include <iostream>

int main() {
std::cout << "Hello World!\n";
return 0;
}

Rust

fn main() {
println!("Hello World!");
}

Go

package main

import "fmt"

func main() {
fmt.Println("Hello World!")
}

JavaScript

console.log("Hello World!");

TypeScript

console.log("Hello World!");

CoffeeScript

console.log "Hello World!"
